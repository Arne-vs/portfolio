Journey of emotion

Wat was de opdracht?

Bij deze opdracht was het de bedoeling dat we zelf vanaf nul een interactieve installatie moesten opbouwen en coderen. We hadden dus veel vrijheid en konden met ons eigen idee komen. Hier ben ik dan op Journey of emotion gekomen.

Wat is journey of emotion?

Dit is een installatie waarmee je jouw meest persoonlijke verhaal, dat door niemand begrepen wordt, kunt visualiseren in de vorm van abstracte kunst. Deze installatie kan niet alleen persoonlijke verhalen weergeven, maar ook visualisaties maken van verhalen over mentale problemen of moeilijkheden, zoals trauma's.

Hoe werkt het?

Via een gecodeerde spraak-naar-tekst functie kan je jouw stem opnemen en omzetten naar tekst. Deze tekst wordt vervolgens via een API in een chatbot van GPT geplaatst, die een korte verhaal prompt schrijft, bedoeld voor een text-to-image functie. Met behulp van de OpenAI API van DALL-E genereert deze text-to-image functie dan een afbeelding die wordt weergegeven. Voor dit proces maak ik voornamelijk gebruik van Next.js en React in JavaScript, samen met mijn terminal.
